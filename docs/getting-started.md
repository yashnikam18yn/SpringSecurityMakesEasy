#  Getting Started

## 1. Add Dependency

```xml
<dependency>
    <groupId>io.github.yashnikam18yn</groupId>
    <artifactId>spring-security-makes-easy</artifactId>
    <version>1.2.0</version>
</dependency>
```

---

## 2. Create Security Config

```java
public class WebSecurity extends EasySecurity {
    // override required methods
}
```

---

## 3. Run Application

Start your Spring Boot application — security is auto-configured 🎉
